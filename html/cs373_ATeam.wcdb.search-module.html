initMatchFound(numTerms, matchFound, orCrises, orPeople, orOrgs, orLis)

source code 

Initializes a dictionary where the keys are model idrefs and the values are lists of booleans. The indices of the lists correspond to a term in the search query. The value at the index is True if that model instance matches that term and False otherwise.

Parameters:
  • numTerms (int) - length of list created by splitting the search query, sent in by the user, by whitespace
  • matchFound (dictionary) - dictionary defined in function description
  • orCrises (set) - Crisis objects whose pages contain matched terms
  • orPeople (set) - Person objects whose pages contain matched terms
  • orOrgs (set) - Org objects whose pages contain matched terms
  • orLis (set) - Li objects whose pages contain matched terms

removeExactLis(exactLis, orSet)

source code 

Function created to deal with duplicate results. All Li instances really just map to some instance of Crisis, Person, or Org. This function accounts for this by removing Li objects whose corresponding model instance has already found matches.

Parameters:
  • exactLis (set) - Li objects with found matches
  • orSet (set) - Crisis, Person, or Org objects with found matches
Returns: set
orSet with the overlap between matched Li objects removed

populate_models(tree)

source code 

Function expects an element tree as a parameter. Main function of the file that calls populate_crisis, populate_org, and populate_person. Returns a dictionary where the keys are the type of model and the values are lists of instances of the models populated by this funciton.

Parameters:
  • tree (ElementTree) - the XML file, uploaded by the user, parsed into an ElementTree
Returns: N/A
function does not return
